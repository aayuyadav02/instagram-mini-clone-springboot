# Instagram Mini Clone (Spring Boot)

A basic full-stack web application built using Spring Boot that demonstrates user signup, login, and simple post creation. The frontend is served using Spring Boot static resources.

## Tech Stack
- Java 17
- Spring Boot
- HTML
- JavaScript

## Features
- User Signup
- User Login
- Create text posts
- View all posts
- Frontend and backend integrated in a single Spring Boot application

## Project Structure
src/main/java
└── controller
└── model
└── service

src/main/resources
└── static
├── index.html
├── feed.html
└── script.js

## How to Run the Project
1. Open the project in an IDE
2. Run `BackendApplication.java`
3. Open browser and visit:
    http://localhost:9090

## Notes
- Data is stored in memory for simplicity
- No database is used
- Focus is on core functionality and frontend-backend integration
